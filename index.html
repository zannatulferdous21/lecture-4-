CSS:There are 3 types of css.
1.inline css.
2.internal css.
3.external css.
IN all these, external css is best to use. But in sometimes we need to use inline css.

An example of inline css:

The process: Inside the body follow the following code

< h3 style="color: green; background-color: yellow;">Lorem ipsum dolor sit amet. < /h3 >

The outcome:

Lorem ipsum dolor sit amet.

An example of internal css:

The process: Inside the head, after title follow the following code

< style >
h4 {
color: blue;
border: green dotted;
} < /style >

The outcome:

Lorem ipsum dolor sit amet.

An example of external css:

The process: you have to creat a file inside of folder css which name will be style.css.Then you have to code the following code in style.css

h1{
color: yellow;
background-color: blue;
font: 20px;
}

The outcome:

Lorem ipsum dolor sit amet.
